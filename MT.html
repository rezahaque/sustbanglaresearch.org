  • Bangla Machine Translation

    This research is aimed on contributing in the machine translation field for English-Bangla language pair. The project is funded by the subproject ‘Development of Multi-Platform Speech & Language Processing Software for Bangla’ (CP-3888) under the 'Higher Education Quality Enhancement Project (HEQEP)' of UGC. The outcome of this research is a machine translation (MT) system which takes English sentence(s) as input and translates them into Bangla sentence(s). The translation engine is acquired by implementing state of the art Statistical model and Neural Network model. For a good quality MT system, large collection of English-Bangla parallel sentences is required. We used around 400,000 parallel sentences to train our language model; some of which were manually generated by human translators and others are acquired from open source corpus. We intend to improve the translation quality by performing hybridization of statistical and neural network model.
